Bonjour ! Je suis JARVIS. Comment puis-je t'aider aujourd'hui ?

Essaie ces commandes :

  • "Raconte-moi une blague" → Pour rire un peu.
  • "Quelle est la date aujourd'hui ?" → Pour connaître la date.
  • "Quelle heure est-il ?" → Pour connaître l'heure.
  • "Calcule 2 + 2" → Pour faire un calcul.
